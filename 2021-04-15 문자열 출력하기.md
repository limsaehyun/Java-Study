문자열 : 쌍따엄표로 둘러싸인 문자들

```
"Hello World!"
"안녕하세요~"
```



**메소드** : 소괄호가 열리고 닫히면 메소드 (나중에 자세히 배움)

​		* 문자열의 출력은 println() 메소드를 사용한다.



```
public class test {
    public static void main(String[] args) {
        System.out.println("이런 식으로 문자열을 출력한다");
    }
}

// 출력 결과: 이런 식으로 문자열을 출력한다.
```
